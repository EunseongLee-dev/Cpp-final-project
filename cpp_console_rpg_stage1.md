## ✅ 핵심 함수 역할 정리

### 1. Attack

-   공격 "결정자"
-   실제 데미지 계산/적용은 하지 않음
-   살아있는지 체크 후 행동 선택

``` cpp
void Attack(Target& target);
```

역할: - 공격 가능 여부 판단 - (추후) 일반공격 / 스킬 랜덤 선택

------------------------------------------------------------------------

### 2. TakeDamage

-   **피해를 받는 쪽에서만** HP 변경
-   사망 여부 판정
-   상태만 변경 (게임 흐름 제어 ❌)

``` cpp
void TakeDamage(int damage);
```

------------------------------------------------------------------------

### 3. Status 출력 함수

-   자기 자신 상태 출력
-   외부 객체를 매개변수로 받을 필요 없음
-   const 멤버 함수로 선언

``` cpp
void Status() const;
```

------------------------------------------------------------------------

## ✅ const 사용 정리 (중요)

### 매개변수 const

``` cpp
void Func(const Player& player);
```

-   전달받은 객체 수정 불가

### 멤버 함수 뒤 const

``` cpp
void Status() const;
```

-   이 함수 안에서 **멤버 변수 수정 불가**
-   출력, 조회용 함수에 사용

### ❌ 잘못된 문법

``` cpp
void const Status(); // 의미 없음 (리턴타입 void는 const 불가)
```

------------------------------------------------------------------------

## ✅ 전방 선언 vs include 정리

### 헤더 (.h)

``` cpp
class Monster;
```

-   포인터 / 참조 매개변수 선언용
-   컴파일 의존성 최소화

### cpp (.cpp)

``` cpp
#include "Monster.h"
```

-   실제 멤버 접근, 함수 호출 시 필요

✅ 헤더 include 제거 → 순환 참조 방지\
✅ cpp include → 구현 가능
